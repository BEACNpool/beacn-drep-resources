# Resource Admission Policy

Only public, reproducible, and citable resources may be admitted.

Required for admission:
- Public URL or committed file path
- Declared schema and format
- Stable identifier (`resource_id`) in `registries/resource_registry.csv`
- License or explicit public-use basis
- Action-type relevance

Disallowed:
- Private chats, DMs, unpublished notes, hidden prompts
- OpenClaw private memory/context files (e.g., local MEMORY.md, assistant journals, local private notes)
- Sources without stable references
- Sources that cannot be independently fetched or replayed

Changes to approved resources must be committed and reviewable.
